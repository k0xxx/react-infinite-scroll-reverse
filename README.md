# COLLEAGAIL MANAGER APPLICATION

## Requirements

- Node ~8
- yarn

---

## Install

```
$ git clone https://gitlab.colleagial.org/k0xxx/colleagial_manager.git
$ cd colleagial_manager
$ yarn install
```

### Configure app

- Create `.env` file

- Fill environment variables

## Simple build for production

```
$ yarn build
```

## For development run

```
$ yarn start
```
